## 🚀 Quick Start (3 Steps!)

### 1️⃣ Extract the ZIP File
Extract all files to a folder of your choice.

### 2️⃣ Open Terminal/Command Prompt
Navigate to the extracted folder:
```bash
cd real_estate_project
```

### 3️⃣ Run These Commands
```bash
# Install dependencies
pip install -r requirements.txt

# Create sample data
python create_sample_data.py

# Start the application
python run.py
```

### 4️⃣ Open Your Browser
Go to: **http://localhost:5000**

## ⚠️ Common Issues & Solutions

### Issue: "Python not found"
**Solution:** Install Python 3.8+ from python.org

### Issue: "Port 5000 already in use"
**Solution:** Stop other applications or change port in run.py

### Issue: "Module not found"
**Solution:** Make sure you ran `pip install -r requirements.txt`
